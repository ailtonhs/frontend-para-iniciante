Tecnologia Web Central

O código HTML é baseado em tags. As tags usam parênteses angulares < >.

        
            <button>

Os sinais de menor e maior < > cercam o nome do elemento que você deseja adicionar à página. Elementos como botões, texto e imagens são adicionado a páginas da web com tags diferentes.

Você pode usar a tag de imagem ( <img> ) para adicionar imagens a uma página web.

        
            <img>

Você precisa de tags HTML para adicionar diferentes elementos a uma página.

  • Botão: <button>
  • Imagem: <img>
  • Parágrafo de texto: <p>
  • Tabela: <table>

O HTML permitirá que você:

  • Crie sites
  • Trabalhe de forma mais eficaz com equipes técnicas
  • Melhore suas habilidades de marketing digital

A estrutura de uma página web é construida em HTML.

Depois você pode estilizar a página com CSS.

JavaScript é usado para tornar as páginas interativas

Conclusões da lição

  • O código HTML controla a estrutura de uma página web
  • As tags HTML são usadas para adicionar elementos a uma página web
  • Alguns exemplos de tags HTML são <button>, <img>, <table> e <p>
