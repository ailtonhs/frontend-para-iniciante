Bem-Vindo ao CSS

CSS é usado para estilizar os elementos em uma página web.

CSS é uma linguagem de estilização que trabalha em conjunto com HTML para dar a uma página sua aparência e layout.

O CSS é construído em cima do HTML. O atributo style permite quevocê use propriedades CSS para personalizar a apresentação visual dos elementos HTML.

        
            <p style="color:green">Texto</p>

A propriedade de cor é usada para controlar a cor do texto.

As propriedades CSS controlam o estilo dos elementos HTML. Exemplos de propriedades CSS são color, border, font-size e margin.

        
            <p style="font-size:large">Texto</p>

As propriedade CSS requerem valores. Valores são configurações possíveis para uma propriedade.

As propriedades e valores CSS são separados com um dois pontos:

        
            <p style="color: navy"> Era uma vez </p>

O código CSS para um elementos HTML deve ser colocado entre aspas duplas ou simples, seguindo o atributo style.

        
            <h1 style="color: blue"> Agenda </h1>

Para aplicar várias propriedade a um elemento, separe as propriedades CSS com um ponto-e-vírgula;

        
            < style="color: yellow; background-color: blue">

CSS significa Cascading Style Sheets e é uma das 3 principais tecnologias. Cascading refere-se ao conjunto de regras que você aprenderá nas próximas lições.
