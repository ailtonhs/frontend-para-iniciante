Código HTML

Código HTML indica ao navegador (por exemplo, Chorme, Safari, etc.) como exibir uma página da web.

Muitos elementos HTML requerem tanto tags de abertura quanto de fechamento.

        
            <button>Like</button>

As tags de fechamento são muitos semelhantes às tags de abertura. A única diferença é que els contêm uma barra inclinada para frente /.

Muitos elementos exigem tanto tags de abertura quanto de fechamento, também conhecidas como tags de contêiner. O texto do parágrafo é outro exemplo.

        
            <p> Era uma vez >/p>

Seu navegador web (por exemplo, Chorme, Safari, ets.) pode ler o código HTML e traduzi-lo no que você vê quando navega na web.

Licões Aprendidas

  • Elementos como botões e textode parágrafo requerem tags de contêiner
  • Tags de contêiner consistem tanto em abertura < > quanto em fechamento </ > tags
