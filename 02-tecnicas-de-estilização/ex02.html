Técnicas de Estilização

Adicionar código CSS a cada elemento HTML leva tempo e torna a estrutura do seu HTML desorganizada.

Quando código CSS é adicionado a elementos HTML, isso é chamado de inline CSS.

        
            <p style="color: red;">Texto do parágrafo</p>

Uma técnica alternativa de estilização é o CSS interno. O CSS interno é usado para estilizar a página inteira. A tag de contêiner <style> é adicionada dentro do documento HTML para agrupar todo o código CSS da página.

        
            <style>
                h2 {
                    color: yellow;
                }
            </style>

O seletor no código CSS corresponde às tags HTML que você deseja alvo.

        
            button {
                color: green;
            }

Separar vários seletores com uma vírgula para aplicar o mesmo estilo a diferentes elementos. Isso torna seu código CSS mais simples.

        
            <style>
                h1, p {
                    color: red;
                    background-color: yellow;
                }
            </style>

É uma boa prática incluir todo o código CSS interno na <head> da página.

        
            <head>
                <style>
                    h1 {
                        color: red;
                        background-color: lime;
                    }

                    button {
                        font-size: 20px;
                    }
                </style>
            </head>

Principais pontos da lição

  • CSS inline é usado para estilizar elemento HTML individuais
  • CSS interno agrupa todo o estilo de uma página dentro da <style> tag
  • Seletores são usado para direcionar elementos HTML
