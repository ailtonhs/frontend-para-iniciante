A Anatomia do CSS

O CSS controla todos os aspectos relacionados ao design da sua página web. Isso inclui fontes, tamanhos, cores, posição, espaçamento, layout, animações e muito mais.

        
            <style>
                p {
                    text-align: center;
                }
            </style>

O código CSS cria regras de estilo. A regra de estilo mais simples consiste em um seletor, mais uma declaração em chaves{}.

        
            button {
                font-size: large;
            }

Uma declaração consiste em duas partes: uma propriedade e um valor.

  • Propriedade: text-align
  • valor: center

Conclusões da Lição

  • O código CSS é composto de regras de estilo
  • Uma regra é composta por um seletor e um conjunto de declarações
  • Uma declaração consiste em um par propriedade: valor
