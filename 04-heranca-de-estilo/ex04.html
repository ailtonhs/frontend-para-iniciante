Herança de Estilo

Com elementos, o elemento externo é conhecido como o parent, e o elemento interno é o child.

        
            <p>
                A programação é uma mistura de <b>lógica</b> e criatividade.
            </p>
  • <p>: parent
  • <b>: child

As regras de estilo para elementos pais também serão aplicadas aos elementos filhos por padrão. Isso é conhecido como herança. Os elementos filhos herdam estilos de seus pais.

A programação é uma mistura de lógica e criatividade.

Por padrão, os filhos hrdam o estilo do seu pai. Se você precisar que um filho tenha um estilo diferente, você deve escrever uma regra separada para o filho.

        
            p {
                color: lightgray;
                background-color: black;
            }

            b {
                color: plum;
                background-color: darkslategrey;
            }

O seletor descendente direciona filhos dentro de um parente especíco. Ele consiste no seletor pai, seguido de um espaço, seguido pelo seletor filho

        
            p, b {
                color: seagreen;
            }

Principais pontos da lição

  • Por padrão, um elemento filho vai herdar o estilo de seu pai
  • Você pode sobrescrever a herança de estilo criando uma regra diferente para o elemento filho
  • O seletor de descendentes visa um filho aninhado dentro de um pai específico
