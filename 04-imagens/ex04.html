Imagens em HTML

As imagens só requerem uma tag para serem exibidas na sua página web

        
            <img src="url">

As imagens não são tecnicamente inseridas em uma página web, elas são vinculadas. A fonte (src) da imagem precisa ser incluída na tag.

Você precisa dizer ao navegador onde encontrar a imagem. A fonte (src) é o local na internet onde a imagem está armazenada.

        
            <img src="http://www.img.jpg">

Navegadores da web solicitam informações da Web para montar e exibir páginas da web.

Códigos, documentos e arquivos de mídia, como imagens e vídeos, são reunidos pelo navegador para exibir a página da web resultante.

Conclusões da lição

  • Imagens podem ser linkadas ou incorporadas em páginas da web
  • Imagens não exigem tags de fechamento
  • A fonte (url) da imagem é necessária para a imagem ser encontrada na internet
