Comentários HTML

Comentários ajudam outras pessoas a lerem o seu código. Você po adicionar notas ou explicações ao seu código com a tag de comentário: <!--...-->

        
            <!--Elemento de parágrafo abaixo-->
            <p> Aqui é um parágrafo </p>

É uma boa prática usar comentários no seu código HTML. Eles explicam o que o código faz.

Os comentários ajudarão outras pesssoas (e você no futuro!) a entender o código.

Você pode usar comentários para desativar temporariamente parte do seu código para que ele não seja exibido pelo navegador.

        
            <!--<button>Ok</button>-->

Quebra de linha

Se você precisar criar uma quebra de linha, pode usar a <br> tag. <br> forçará uma quebra de linha.

        
            <p>Primeira linha do texto.<br>
            Segunda linha do texto.</p>

Lições Aprendidas

  • Comentários tornam seu código mais fácil de ler e entender
  • A tag <!--..--> permite que você adicione comentário ao seu código
  • A tag <br> é usada para criar quebras de linha
