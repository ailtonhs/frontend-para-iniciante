Seletores de ID e classe

Seletor ID

Para controle total sobre o design dos seus elementos HTML, você pode dar a eles IDs e classes.

O atributo id é usado para dar um identificador único a um elemento específico na página.

        
            <p id="intro">Tenha um ótimo momento de cinema!</p>

Você pode direcionar elementos específicos usando hashtag # seguido pelo nome do ID como seletor.

        
            #intro {
                color: black;
                background-color: green;
            }

Seletor de classe

Outra maneira de identificar elementos é o atributo de ckasse. Use o atributo de classe para dar o mesmo identificador a um grupo de elementos.

        
            <p class="filme"> Avatar </p>

            <button class="filme"> Assistir agora </button>

Você pode criar regras de estilo para um grupo de elementos na mesma classe (grupo).

Use um ponto . seguido pelo nome da classe para direcionar todos os elementos na mesma classe.

        
            .filme {
                color: white;
                background-color: navy;
            }

Você também pode direcionar apenas aqueles elementos de um tipo específico que possuem uma determinada classe.

        
            p.filme {
                font-size: 18px;
            }

Principais pontos da lição

  • O seletor ID usa o símbolo hash # e direciona a um elemento específico
  • Uma classe fornece um identificador comum a um grupo de elementos
  • O seletor de classe usa um ponto . e direciona a um grupo de elementos com a mesma classe
