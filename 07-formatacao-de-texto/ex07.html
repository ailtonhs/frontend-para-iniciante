Formatação de Texto

O HTML tags de formatação são usadas para alterar como o texto é exibido.

A tag negrito <b> é usada para exibir texto em negrito.

        
            <b>Web development</b>

As tags de formatação são tags de contêiner. Isso significa que ambas, tags de abertura e fechamento, são necessárias.

A tag itálica <i> é utilizada para exibir texto em itálico.

        
            <i>Digital marketing</i>

A tag de sublinhado <u> é usada para sublinhar o texto.

        
            <u>Search Engine</u>

Existem algumas tags de formatação HTML que você pode usar para tornar suas páginas da web mais acessíveis.

A tsg forte <strong> é usada para destacar importante texto.

        
            <strong>Warning!</strong>

<strong> o texto é exibido em negrito, assim como você usa a tag <b>. A diferença é que a tag <strong> também tem significado e é usada por leitores de tela.

A tag de ênfase <em> é usada para definir texto enfatizado.

        
            <em>The Book of Life</em>

O texto enfatizado é exibido em itálico, assim como quando você usa a tag <i>. A diferença é que o leitor de tela vai enfatizar verbalmente as palavras.

Principais pontos da lição

  • Você pode usar aninhamento para tornar o texto negrito, itálico e sublinhado com as tag <b>, <i>e <u> tags de formatação
  • Você pode tornar suas páginas da web mais acessíveis com o uso de <strong> e <em> tags de formatação semântica
