HyperText

HyperText é um texto que contém um link para outra página.

Páginas da Web são chamadas de documentos HyperText porque estão conectadas por links de HyperText (ou hiperlinks).

A tag âncora <a> é usada para criar um hyperlink numa página Web.

A tag <s> é uma tag de contêiner que requer tanto a tag de abertura quanto a de fechamento.

Os hiperlinks são usados para linkar de uma página web para outras. Para criar um link, você precisa de "href" para adicionar a URL de destino.

        
            <a href="url">Z</a>

Você também pode usar hiperlinks para conectar páginas do mesmo site.
