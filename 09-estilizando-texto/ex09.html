Estilizando Texto

Vamos começar com o alinhamento de texto.

        
            h1 {
                text-align: center;
            }

Use text-align: justify para alinhar o texto tanto à esquerda quanto à direita, ajustando o espaçamento entre as palavras, garantindo que cada linha tenha a mesma largura

        
            p {
                text-align: justify;
            }

Você pode adicionar text-decoration para transmitir significado ou chamar a atenção para partes específicas do texto, como links.

        
            h3 {
                text-decoration: underlined;
            }

            h4 {
                text-decoration: overlined;
            }

            h5 {
                text-decoration: line-through;
            }

            h6 {
                text-decoration: underlined overline
            }

O texto pode ter vários estilos de decoração, como pontilhado, duplo, tracejado e ondulado.

A propriedade text-decoration é uma forma curta e simples de se referir a diferentes sub-propriedades, como text-decoration-line, text-decoration-color e text-decoration-style

        
            p {
                text-decoration: wavy overline #2A9D8F;
            }
  • text-decoration-line: wavy
  • text-decoration-color: overline
  • text-decoration-style: #2A9D8F

A propriedade text-transform permite que você controle a capitalização do conteúdo do texto. Ele pode receber 3 valores capitalize, uppercase e lowercase.

O text-shadow cria um efeito de profundidade, dá ênfase, ou simplesmente adiciona um toque elegante à sua tipografia.

O deslocamento horizontal é o quando para a direita (valores positivos) ou para esquerda (valores negativos).

O deslocamento vertical é o quanto para baixo (valores positivos) ou para cima (valores negativos).

O text-shadow pode aceitar dois valores adicionais e opcionais.

  • blur radius: a quantidade de desfoque aplicada à sombra
  • color: a cor da sombra
        
            h1 {
                color: #FFFFFF; /* white colored text */
                text-shadow: 15px 12px 5px #4296CE; 
                font-size: 4rem;
              }
  • deslocamento horizontal: 15px
  • deslocamento vertical: 12px
  • cor: #4296CE
  • raio de desfoque: 5px

Principais Lições

  • text-align controla o alinhamento horizontal do texto
  • text-decoration adiciona efeito decorativos ao texto
  • text-transform altera a capitalização do texto
  • text-shadow aplica efeitos de sombra atrás do texto
