Listas HTML

Uma lista consiste em itens de lista <li>.

        
            <li> Item 1</li>

            <li> Item 2</li>

            <li> Item 3</li>

Itens da lista são tags de contêiner.

Listas ordenadas

Lista ordenadas <ol> são exibidas com números (1,2,3..) em vez de pontos de marcador. Use <ol> quando os pontos têm uma ordem específica, como instruções passo a passo

        
            <ol>
                <li> Café </li>
                <li> Chá </li>
                <li> Leite </li>
            </ol>

As listas também são tags de contêiner, com os itens da lista aninhados dentro.

Lista não ordenada

Use lista não ordenada <ul> quando a ordem dos itens não é importante.

        
            <ul>
                <li> Café </li>
                <li> Chá </li>
                <li> Leite </li>
            </ul>

Conclusões da Lição

  • As listas são feitas de itens de lista <li>
  • Você pode adicionar listas ordenadas <ol> e listas não ordenadas <ul> às páginas da Web
  • Você pode aninhar listas dentro de outras
