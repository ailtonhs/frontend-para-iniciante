Atributos HTML

Atributos podem ser opcionais ou obrigatórios. Você já conhece alguns atributos obrigatórios.

A tag de imagem requer um atributo para funcionar corretamente.

        
            <img src="url">

Para tags de contêiner, os atributos sempre vão na tag de abertura. A tag âncore é outro exemplo de um elemento HTML que requer um atributo para funcionar corretamente.

        
            <a href="url">Z</a>

É uma boa prática descrever o que as imagens estão mostrando. O atributo alt (texto alternativo) é usado para adicionar descrições de imagens.

        
            <img src="url" alt="Descrição da imagem">

O atributo alt pode ser:

  • Lido em voz alta por leitores de tela
  • Exibido quando a imagem não carrega
  • Lido por motores de busca

Você pode controlar o tamnaho das imagens em suas páginas da web. Width é uma atributo opcional.

Height é um atributo opcional para o elemento de imagem.

Quando apenas um dos atributos é dado, o navegador web calculará o outro com base na proporção de aspecto para que a imagem não seja esticada ou comprimida.

Principais pontos da lição

  • Você pode adicionar atributos aos elementos HTML para fornecer informações adicionais
  • Alguns atributos são necessários para o elemento funcionar corretamente
  • Atributos são sempre especificados na tag de início
