Estilos de Fontes

As fontes desempenham um papel crucial noo design da web, influenciando a legibilidade, estética e experiência do usuário. Elas dão vidas às palavras numa página e define o tom do conteúdo.

A propriedae font-family

A propriedade font-family em CSS permite especificar o tipo de letra para o seu texto. Isso determina como o texto aparece em uma página da web.

        
            p {
                font-family: Arial;
            }

Muitas fontes são instaladas por padrão em todos os dispositivos. Estas são conhecidas como fontes web safe.

Outras fontes podem precisar ser instaladas no dispositivo do usuário, então você precisará especificar um fonte web safe como um back-up, caso o usuário não as tenha.

A propriedade font-family especifica uma lista de fontes, da maior propriedade para a menor. A seleção de fontes para na primeira fonte da lista que está no sistema do usuário.

        
            p {
                font-family: Helvetica, Georgia, Arial;
            }

Famílias de fontes genéricas são grupos de fontes com tipos de letra semelhantes, e são usadas como fallbacks universais. É uma melhor prática colocá-las como última opção em uma propriedade de font-family, Se as fontes preferidas não estiverem disponíveis, os navegadores usarão uma fonte dessa família.

Use aspas para envolver nomes de fontes com várias palavras. É uma boa prática.

        
            p {
                font-family: "Courier New", monaco, monospace;
            }

Font-weight

A propriedade font-weight controla a espessura(ou negrito) do texto.

        
            .button {
                font-weight: 500;
            }

Você pode usar valores númericos de 100(mais fino) para 900(mais espesso) para especificar o peso da fonte, aumentando em incrrementos de 100.

        
            #light {
                fonte-weight: 100;
            }

            #normal {
                font-weight: 400;
            }

            #bold {
                font-weight: 700;
            }

            #boder {
                font-weight: 900;
            }

Certos valores de peso, como 400 e 700, têm equivalentes nomeados que podem ser usados diretamente como valores: normal e bold.

  • normal: 400
  • bold: 700

Conclusões da Lição

  • font-family permite especificar o tipo de letra
  • Você pode usar font-weight para controlar a espessura do texto
