Estilizando Links

Os links são a essência da web, conectando uma página a outra. Eles são cruciais para navegação, mas o estilo padrão sublinhado em azul nem sempre se ajusta a todos os designs.

Você pode substituir este estilo padrão com CSS para personalizar cores, fontes e decoradores.

        
            a {
                color: #52C8aC;
                font-family: 'Georgia', serif;
                background-color: #D6E5E3;
            }

Quando os usuários interagem com um link (também outros elementos), ele muda seu estado. Você pode se referir ao status de um elemento com pseudo seletores

        
            //Define o estilo de links não visitados
            a:link {
                color: #d11bce;
                text-decoration: none;
            }

            //Um link muda seu estado quando o mouse está sobre ele
            a:hover {
                color: #6897ee;
                background-color: #000000;
            }

            //Clicou no link mas não soltei o botão do mouse ainda
            a:active {
                color: #b8860b;
                background-color: #000000;
            }
Link

Por padrão, todos os links são sublinhados. Você pode remover o estilo de sublinhado definindo a propriedade test-decoration pa none.

Conclusões da Lição

  • Você pode usar seletores pseudo para estilizar os diferentes estados dos links
  • Você pode substituir o formato de sublinhado padão com a propriedade text-decoration
