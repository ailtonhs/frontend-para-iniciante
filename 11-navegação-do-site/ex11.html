Navegação do Site

A forma com os visitantes navegam em um site depende do seu design. Os sites vêm em 2 formas:

  • Sites de várias páginas
  • Sites de uma única página

Tag de contêiner <nav>

A tag de contêiner <nav> define um conjuto de links que permite ao usuário navegar entre as páginas de um site.

        
            <nav>
                <a href="index.html">Home</a>
                <a href="about.html">About</a>
                <a href="contact.html">Contact</a>
            </nav>

Os links para as diferentes páginas são adicionados com a tag de âncora <a> e aninhados dentro da tag do contêiner <nav>

Projeto HTML

O projeto HTML para um site de várias páginas é composto de diferentes documentos HTML (ou arquivos).

Página Única

Um site de uma única página tem todos o seu conteúdo na página inicial. Qualquer link de navegação leva o visitante para diferentes seções(em vez de carregar noas páginas).

Para saltar para uma parte específica de um site de página única, primeiro você precisa marcar a seção com o atributo id(ID).

        
            <h2 id="section-1">Section 1</h2>

Uma vez que o elemento ao qual você deseja saltar tenha sido marcado com um id, você pode direcioná-lo com uma tag de âncora <a>.

        
            <a href="#s1">Ir para S1</a>

            <h2 id="s1">Seção 1</h2>

Conclusões da Lição

  • Os sites vêm em 2 tipos diferentes: de página única ou de várias páginas
  • Você pode criar links de navegação como tags <a> dentro de uma tag <nav>
  • Você pode vincular a partes específicas de uma página, dando-lhes um atributo id
