Formulários HTML

Você pode interagir com os visitantes do seu site e coletar informações deles usando forms.

Você pode usar formulários para permitir que seus visitantes:

  • Entrem em contato com você
  • Enviem pedidos, solicitações e outras informações
  • Criem uma conta ou se registrem
  • e muito mais

Os formulários são compostos por elementos de entrada como campos de texto, caixas de seleção e botões de envio. Esses elementos de entrada estão aninhados dentro da tag de contêiner <form>.

        
            <form>
                <!--os elementos de entrada vão aqui-->
            </form>

Elementos <input>

O elemento de formulários mais comum é <input>. Exitem muitas formas do elementos <input>, dependendo do atributo type.

Os elementos do formulário serão exibidos na mesma linha, a menos que usemos a tag <br>(quebra de linha).

Rótulos podem ser adicionados para os diferentes elementos de entrada com a tag <label>.

        
            <form>
                <label>email:</label>
                <input type="text">
            </form>

É considerado uma boa prática conectar o label aos elementos de input através do uso dos atributos for e id.

O atributo id é usado para identificar um elemento de entrada única. O atributo for em uma etiqueta direciona (e combina!) com o id de um input.

        
            <label for="r1">Texto</label>

            <input type="text" id="r1">

Lições Aprendidas

  • Você pode adicionar formulários às suas páginas web com a tag de contêiner <form>
  • Os formulários são compostos de elementos <input>
  • Você pode etiquetar as diferentes entradas em um formulário com a tag de contêiner <label>
