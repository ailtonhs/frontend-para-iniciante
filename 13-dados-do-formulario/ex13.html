Dados do Formulário

Um formulário HTML é uma maneira conveniente de enviar dados para um banco de dados hospedado em um servidor.

Um botão de submite é usado para enviar os dados em um formulário. O tipo de entrada submite adiciona um botão ao formulário.

        
            <form>
                <label for="t1">Email</label>
                <input type="text" id="t1">

                <label for t2="t2">Country</label>
                <input type="text" id="t2">

                <input type="submit">
            </form>

Você pode usar a entrada submit para enviar os dados do formulário para um banco de dados hospedado em um servidor.

O atributo name é usado para referenciar os dados após o envio do formulário

        
            <form>
            <label for="t1">Email</label>
            <input type="text" name="email" id="t1">

            <label for t2="t2">Country</label>
            <input type="text" name="cidade" id="t2">

            <input type="submit">
        </form>

Apenas elementos de formulário com um atributo name terão seus valores passados para o banco de dados ao enviar um formulário.

Principais liçoes

  • O tipo de entrada submit permite que o usuário envie dados do formulário
  • O atributo name é usado para dizer ao banco de dados onde armazenar os dados
  • O atributo value é usado para dizer ao banco de dados o que armazenar
