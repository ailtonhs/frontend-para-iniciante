Menu Drop-Down

Menus suspensos tornam seus formulários mais eficientes, acessíveis e organizados.

Você pode usar a <select> tag de contêiner para criar uma lista suspensa.

        
            <select>
                <!--opções-->
            </select>

A tag de container <option> é usada dentro de uma tag <select> para adicionar opções para o usuário.

        
            <select>
                <option> Opt 1 </option>

                <option> Opt 2 </option>

                <option> Opt 3 </option>
            </select>

Você pode usar o elemento <select> como parte de um formulário para coletar entrada de dados do usuário.

        
            <form>
                <select>
                <option> Opt 1 </option>

                <option> Opt 2 </option>

                <option> Opt 3 </option>
                 </select>
            </form>

Enviar um formulário envia informações para um banco de dados. O atributo name é necessário para enviar os dados do formulário.

        
            <form>
                <select name="color">

                <option> Red </option>

                <option> Blue </option>

                </select>
             </form>

Conclusões da lição

  • A tag <select> é usada para criar um menu suspenso
  • A tag <option> é usada para dar opções ao usuário
  • Você pode usar menus suspensos em formulários, em combinação com outros elementos de entrada
