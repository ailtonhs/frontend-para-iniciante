Vídeos HTML

Você pode usar a tag de contêiner <video> para incorporar arquivos de vídeos em uma página da web.

        
            <video>
                <!--arquivo de vídeo-->
            </video>

Você pode adicionar arquivos de vídeos em diferentes formatos. Os formatos de vídeo comuns são: MP4, OGG e WebM.

        
            <video>
                <source src="file.mp4" type="video/mp4">
            </video>

A tag vazia <source> fornece a localização e o formato do arquivo de vídeo.

O atributo src especifica a URL do arquivo de vídeo.

O atributo type especifica o formato do arquivo de vídeo.

Diferentes formatos de mídia são necessário para compatibilidade com diferentes navegadores e dispositivos. A tag <source> permite que você adicione vários arquivos.

        
            <video>
            <source src="file.mp4" type="video/mp4">

            <source src="file.webm" type="video/webm">
            </video>

Quando diferentes opções de <source> são incluídas, o navegador escolherá a primeira que ele suporta.

Você pode adicionar texto entre as tags <video>. O texto só será exibido em navegadores que não suportam o elemento de vídeo.

        
            <video>
            <source src="file.mp4" type="video/mp4">

            <source src="file.webm" type="video/webm">
            </video>

            Video não suportado

Você pode exibir play/pause, volume e outros controles de vídeos com o atributo controls.

        
            <video controls>
            <source src="file.mp4" type="video/mp4">

            <source src="file.webm" type="video/webm">

            Vídeo não suportado!
            </video>

Lições Aprendidas

  • A tag <video> incorpora vídeo nas páginas da web
  • A tag <source> permite que você adicione vários arquivos de mídia
  • O atributo controls adiciona controles de vídeo padrão.
