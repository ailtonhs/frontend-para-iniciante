Mídias de Áudio

Você pode incorporar arquivos de áudio em páginas da web com a <audio> tag de contêiner.

        
            <audio>
                <!--arquivo de áudio-->
            </audio>

Assim como o vídeo, a <source> tag é usada para adicionar opções de fonte para áudio.

        
            <audio>
                <source src="audiofile.mp3" type="audio/mp3">

                <source src="audiofile.ogg" type="audio/ogg">
            </audio>

Assim como o vídeo, o src atributo adiciona a URL do arquivo de áudio. O type atributo adiciiona o formato.

Assim como o vídeo, você pode adicionar arquivos de áudio em diferentes formatos. Os formatos de áudio comuns são: MP3, OGG e WAV.

Você pode usar os atributos autoplay, muted e loop para controlar o comportamento do elemento multimídia. Assim como controls, esses atributos têm seus valores padrão omitidos.

  • Sem som: muted
  • Começa a tocar automaticamente: autoplay
  • Toca em repetição: loop
  • Mostra controles de mídia: controls

Os atributos autoplay, muted e loop podem ser usados tanto em elementos de vídeo quanto de áudio.

Principais Conclusões da Lição

  • A tag <audio> incorpora som em uma página da web
  • Os atributos autoplay, muted e loop controlam como o multimídia se comporta
