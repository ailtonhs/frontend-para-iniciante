Layout da Página

O corpo de uma página web pode ser dividido em 3 partes.

  • Cabeçalho: header
  • Conteúdo principal: main
  • Rodapé: footer

A tag de contêiner <header> geralmente contém informações introdutórias.

        
            <header>
                <!--Informação introdutória-->
            </header>

O cabeçalho geralmente contém links de navegação, um menu e/ou uma barra de pesquisa. Elementos da marca como logotipos também são geralmente encontrados no cabeçalho.

A tag de contêiner <main> é usada para incluir o conteúdo principal de uma página web.

        
            <main>
                <!--Conteúdo principal-->
            </main>

A tag de contêiner <footer> geralmente contém informações de contato e legais e links.

        
            <footer>
                <!--Informação do rodapé-->
            </footer>

Os elementos <header>, <main> e <footer> estão aninhado dentro da tag de contêiner <body>.

        
            <body>

                <header>
                    informações introdutórias
                </header>

                <main>
                    conteúdo principal
                </main>

                <footer>
                    Informação do rodapé
                </footer>

            </body>

Principais Lições

  • Podem ser usados para criar um layout de página em 3 partes
  • São tags de contêiner
  • Estão aninhados dentro do corpo
