Elementos Inline e Block

Cada elementos HTML é ou inline ou block.

Elementos de nível de bloco sempre começam em uma nova linha. Elementos inline não começam em uma nova linha.

Elementos de nível de bloco ocupam toda a largura disponível. Elementos inline só ocupam tanto de largura quanto necessário.

Este código resultará em 2 botões na mesma linha, por quê são elementos inline.

        
            <button>Botão 1</button>
            <button>Botão 2</button>

Elemento inline podem ser aninhados dentro de elementos de nível de bloco.

        
            <p>
                <a href="#">Link</a>
            </p>

Link

As tags de quebra de linha <br> são usadas para forçar os elementos inline a começar em uma nova linha

        
            <button>Botão A</button><br>
            <button>Botão B</button>

Conclusões da Lição

  • Exitem dois tipos de elemento HTML: inline e block
  • Elemento de nível de bloco sempre começam( e termina) em uma nova linha
  • Elementos inline não começam em uma nova linha
