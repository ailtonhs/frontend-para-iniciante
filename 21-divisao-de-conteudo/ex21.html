Divisão de Conteúdo

Agrupar diferentes elementos HTML pode tornar suas páginas mais rápidaa para carregar e mais fáceis de personalizar e manter.

O <div> elemento é um recipiente para elementos HTML que mantém suas páginas organizadas

        
            <div>
            <!--Elemento HTML 1 -->

            <!--Elemento HTML 2 -->
            </div>

O elemento <div> sempre ocupa a largura total disponivel. Isso significa que o elemento <div> é nível de bloco.

        
            <div>

            <h1> Cabeççalho </h1>

            <p> Parágrafo  </p>

            <img src="url">
            
            </div>

<div> é um container que agrupa conteúdo relacionado em uma página web, como uma barra lateral ou um menu de navegação. <div> não adiciona significado ao conteúdo porque é uma tag não-semântica.

O estilo em um contêiner <div> será aplicado a todos os seus elementos aninhados, a menos que você os forneça os seus próprios.

        
            <div style="color: rede">

            <h2> Título 2 </h2>

            <h3 style="color: green"> Título 3 </h3>

            <p> Parágrafo  </p>

            </div>

Conclusões da Lição

  • <div> div é um recipiente que agrupa elemento
  • <div> é um elemento de nível de bloco
  • <div> é usada para aplicar o mesmo estilo a elementos agrupados
