Tabelas HTML

As tabelas ajudam a exibir dados de uma maneira que é fácil de escanear, comparar e analizar.

Você pode adicionar uma tabela à sua página web com a tag do recipiente <table>.

        
            <table>
            <!--linha e coluna-->
            </table>

Você pode adicionar linhas a uma tabela com a tag de contêiner <tr> (table row). As linhas estão aninhadas dentro da tag <table>.

        
            <table>
            <tr>
            <!--dados da linha-->
            </tr>
            </table>

Você pode adicionar células com a tag de contêiner <td>

        
            <table>
                <tr>
                    <td> Célula 1</td>
                    <td> Célula 2</td>
                </tr>
            </table>

Tabela geralmente inclue cabeçalho. Um cabeçalho é uma linha especial no topo da tabela usada para rotular cada coluna.

Você pode criar células de cabeçalhp com <th>

        
            <table>
                <tr>
                    <th>Nome</th>
                    <th>Idade</th>
                </tr>

                <tr>
                    <td>Amy</td>
                    <td>38</td>
                </tr>
            </table>

Você pode criar célula que ocupam várias linhas e/ou colunas, usando os atributos colspan e rowspan. Isso é chamado de mesclagem de células

Colspan é para o número de colunas que uma célula deve abranger.

        
            <table>
                <tr>
                    <th>Month</th>
                    <th>Budget($)</th>
                <tr>

                <tr>
                    <td>January</td>
                    <td>500</td>
                <tr>

                <tr>
                    <td>February</td>
                    <td>400</td>
                <tr>

                <tr>
                    <td colspan="2">Sum: 900</td>
                <tr>
            </table>
Month Budget($)
January 500
February 400
sum: 900

O atributo rowspan especifica o número de linhas que uma célula deve abranger. Você pode usar rowspan para mesclar células verticalmente.

        
            <table>
            <tr>
                <th>Month</th>
                <th>Budget($)</th>
                <th>Savings($)</th>
            <tr>

            <tr>
                <td>January</td>
                <td>500</td>
                <td rowspan="2">150</td>
            <tr>

            <tr>
                <td>February</td>
                <td>400</td>
            <tr>

        </table>
Month Budget($) Savings($)
January 500 150
February 400

Conclusões da Lição

  • Você pode adicionar tabelas às páginas da web com a tag de contêiner <table>
  • A tag de contêiner <tr> é usada para adicionar linhas
  • A tag de contêiner <td> é usada para adicionar dados
  • A tag de contêiner <th> é usada para adicionar cabeçalhos
